- What Is AI Agent Memory? | IBM: # What is AI agent memory? AI agent memory refers to an artificial intelligence (AI) system’s ability to store and recall past experiences to improve decision-making, perception and overall performance. AI agents do not necessarily need memory systems. Long-term memory (LTM) allows AI agents to store and recall information across different sessions, making them more personalized and intelligent over time. Semantic memory is responsible for storing structured factual knowledge that an AI agent can retrieve and use for reasoning. AI agents typically implement semantic memory using knowledge bases, symbolic AI or vector embeddings, allowing them to process and retrieve relevant information efficiently. Procedural memory in AI agents refers to the ability to store and recall skills, rules and learned behaviors that enable an agent to perform tasks automatically without explicit reasoning each time. ## Frameworks for agentic AI memory.
